教师作业要求如下
使用mixup、mixmatch、fixmatch中的两种方式实现基于半监督学习的sar地面目标识别
用搭建的网络训练我的数据集，得到训练模型并在测试集上进行测试。
搭建界面系统，具体能够读入图片并显示检测结果整体
具体实现代码完整全部
sar-ground-target-recognition/
│
├── data/
│   ├── train/              # 训练集图片
│   ├── test/               # 测试集图片

│

│
├── models/                 # 定义神经网络模型
│   ├── network.py          # SARNet 网络结构
│
├── app/                    # Web 界面相关代码
│   └── app.py              # Flask 应用
│
├── main.py                 # 主训练和测试脚本
├── requirements.txt        # Python 环境依赖
├── readme.md               # 项目说明文档
└── utils.py                # 辅助工具函数，如数据加载、日志等
